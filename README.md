# 🤖 Persuation Bot
An LLM-based chatbot that takes in messages from a user, processes them, and generates responses intended to defend itself or maintain its position during an ongoing conversation.

## Architecture
![Architecture Diagram](docs/architecture.png?v=2   )

- **Client**: Anyone consuming the API.
- **API**: REST API built with FastAPI.
- **Database**: Volatile Inmemory storage for message storage.
- **Cache**: Redis Cache to improve Latency and Rate Limiting

### Caching Strategyy
We cache three things to improve latency, cut costs, and avoid duplicate work:

**Idempotency Keys**

- What: A short-lived record of processed POST requests.

- Why: Prevents double processing when clients retry or double-send.

- Key: idempotency:{request_uuid}
TTL: 5–15 min.

**Last Messages** (Conversation History Cache)

- What: The last N messages for a conversation.

- Why: Builds LLM context without hitting the database each turn.

- Key: conversation:{conversation_id}:history

- TTL: 30–60 min (reset on activity).

**LLM Reply Cache**

- What: The model’s reply for an identical prompt/context.

- Why: Skips repeat LLM calls on retries or duplicate user inputs.

- Key: llm:{sha256(model + system_prompts + subject + side + last_K_msgs + user_msg + temperature)}

- TTL: 1–24 h (depends on how stable you want outputs to be).

## Entities
![Entities](docs/entities.png)

Messages will contain ai prompts and responses.

- ```id```: id autogenerated
- ```conversation_id```: related conversation
- ```message```: message content
- ```role```: message owner (user, bot)
- ```timestamps```

Conversations will be used to link every message based on topic

- ```id```: id autogenerated
- ```side```: wheter bot will be against or in favor of (side, pro)
- ```topic```: subject of a conversation or discussion.
- ```timestamps```

## Example Requests

**POST /messages**
```http
POST /messages HTTP/1.1
Host: http://localhost:8000
Content-Type: application/json
```

```
{
    "conversation_id": null,
    "message": "Topic: Sports are one of the best ways to build discipline and strengthen community bonds. Side: PRO."
}
```
**Response 201**
```
{
    "conversation_id": "1",
    "message": [
        {
            "role": "user",
            "message": "Topic: Sports are one of the best ways to build discipline and strengthen community bonds. Side: PRO."
        },
        {
            "role": "bot",
            "message": "I will gladly take the PRO side. Sports are key in shaping discipline..."
        }
    ]
}
```

POST /messages HTTP/1.1
